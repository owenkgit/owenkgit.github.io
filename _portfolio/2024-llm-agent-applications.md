## Overview
Developed various automated services leveraging OpenAI's GPT model, including ad copy generation, document summarization, and virtual assistant functionalities.

## Features
- **Ad Copy Generation**: Generate tailored ad copy based on product details and brand tone
- **Document Summarization**: Multi-language document summarization service
- **Content Creation**: Blog posts and Instagram captions
- **PDF Summarization**: Extract concise summaries from PDF documents
- **Telegram Bot**: Automated services and interactions
- **Virtual Assistant**: Task handling including scheduling and reminders
- **YouTube Summarization**: Quick video insights

## Technologies
Python, Streamlit, OpenAI API

## Overview
Contributed to the development and dissemination of open source tools and datasets for BCI research, providing foundational resources for the Brain-Computer Interface community.

## Open Dataset
**Link**: [gigadb.org/dataset/100542](https://gigadb.org/dataset/100542)

### Dataset Specifications
- **Participants**: 54 healthy subjects (ages 24-35; 25 females)
- **Data Quality**: 64-channel EEG across multiple sessions
- **Duration**: 5-6 hours per subject per experiment
- **Paradigms**: Motor Imagery, Event-Related Potential (P300), and Steady-State Visually Evoked Potential (SSVEP)

## Open Source
**GitHub**: [github.com/PatternRecognition/OpenBMI](https://github.com/PatternRecognition/OpenBMI)

### Features
- Various BCI experiments (Binary-class MI, 36 symbol ERP speller, Four target frequencies SSVEP)
- Full open-source package from study design to outcome
- Advanced pattern recognition algorithms
- Analysis tools for BCI users

## Contributions
- Developed open-source project with various BCI paradigms and analysis functions
- Managed experimental process during open-dataset project
- Part leader in team of 4 contributors
- Published as 2nd author in GigaScience journal

## Impact
- One of the largest datasets in BCI fields
- High-quality data enabling reproducible research
- Easy and useful analysis tools for BCI researchers
- Supports research into BCI illiteracy and future directions
